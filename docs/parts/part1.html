Partial Application and Pattern Matching

Duration

10 minutes

Lab Goals

The primary goal of this lab will be to start working with F# option types to sum a list of numbers.

  1. First, create an F# library project and open the script.fsx file.
  2. Copy the the list of inteter option types and sum the values.
  3. Run the code in the REPL.

Steps

Option Types

  1. Start a new F# library project and open the script.fsx file.
  2. Make sure your REPL window is open and has been reset.
  3. Paste the following list into the script file:
    4. [None;
Some(147);
Some(12);
Some(38);
None;
Some(41);
Some(59);
Some(60);
Some(73);
None;
Some(27);
Some(43);
Some(15);
None;
Some(35);
Some(4);
Some(10);
Some(15)]
  4. Filter the list of integer option types using the List.filter function with the IsSome function to create a list that only contains options with values.
  5. Use the List.map function with the Value function to create a list of integers from the option type values.
  6. Sum the resulting list.

Show Code 

   |> List.filter (fun x -> x.IsSome)
   |> List.map (fun x -> x.Value)
   |> List.sum

Summary

In this lab, we worked option types and extracted the values from a list of option types containing integers.
