Before you begin

This is a F# language class. To complete the coding exercises, you will need a Xamarin development environment (Windows or macOS) to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In this class we will be exploring the functional aspects of F# and why you should prefer them over the object-oriented features inherited from .NET. We will also be looking at how to use one of the most powerful features of F# - pattern matching.

Lecture

FSC104 - Partial Application and Pattern Matching.

Objectives

  1. Examine functional aspects of F#
  2. Employ pattern matching
  3. Apply active patterns

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Utilizing option types to sum numbers
    Exercise
    In this group exercise, we will sum a set of option types together.
  2. Applying partial application
    Exercise
    In this exercise, you will build a generic distance function to calculate the distance between two points.
  3. Practice pattern matching
    Exercise
    In this group exercise, you will build a generic distance function to calculate the distance between two points.
  4. Apply pattern matching to create a daily routine
    Exercise
    In this exercise, you will generate a daily schedule for a 31-day month which will output your breakfast and commute route.
  5. Utilizing active patterns to create a flight itinerary
    Exercise
    In this exercise, you will utilize active patterns to calculate which airline had the lowest domestic deplaning count in SFO for December of every year.
